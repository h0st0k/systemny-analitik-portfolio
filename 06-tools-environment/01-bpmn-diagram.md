# BPMN-диаграмма: Процесс бронирования книги в библиотеке

```mermaid
graph TD
    Start((Начало)) --> Step1[Читатель ищет книгу в каталоге]
    Step1 --> Condition1{Книга доступна?}
    Condition1 -->|Да| Step2[Читатель нажимает Забронировать]
    Condition1 -->|Нет| Step3[Показать сообщение: Книга недоступна]
    Step3 --> End((Конец))
    
    Step2 --> Condition2{Лимит броней < 3?}
    Condition2 -->|Да| Step4[Создать бронь на 3 дня]
    Condition2 -->|Нет| Step5[Показать сообщение: Превышен лимит броней]
    Step5 --> End
    
    Step4 --> Step6[Отправить уведомление читателю]
    Step6 --> End

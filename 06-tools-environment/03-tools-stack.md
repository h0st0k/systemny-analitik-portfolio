# Мой инструментальный стек системного аналитика

## 📝 Фиксация требований
| Инструмент | Назначение | Как использую |
|------------|------------|---------------|
| **GitHub Issues** | Трекинг задач | Хранение user stories и acceptance criteria |
| **Markdown** | Документирование | Все требования и артефакты в .md файлах |

## 📊 Моделирование
| Инструмент | Назначение | Как использую |
|------------|------------|---------------|
| **Mermaid** | Диаграммы в коде | Sequence, ER, Context diagrams |
| **Draw.io** | Сложные схемы | BPMN, архитектурные схемы |
| **PlantUML** | Кодовые диаграммы | Планирую освоить |

## 📡 API и интеграции
| Инструмент | Назначение | Как использую |
|------------|------------|---------------|

| **Postman** | Тестирование API | Авторская приемка |

| Инструмент | Назначение | Как использую |

# Критерии приёмки (Acceptance Criteria)

## Сценарий: Голосовое ведение маршрута

| ID | Критерий | Статус |
|----|----------|--------|
| AC-01 | При выборе велосипедного/самокатного маршрута голосовое ведение включается автоматически | ✅ |
| AC-02 | Алиса озвучивает: "Через 200 метров поверните направо" | ✅ |
| AC-03 | Подсказки работают при свёрнутом приложении (фон) | ✅ |
| AC-04 | При отклонении от маршрута голос говорит: "Вы сбились с маршрута, перестраиваю..." | ✅ |
| AC-05 | Пользователь может отключить голос в любой момент (кнопка в интерфейсе) | ✅ |
| AC-06 | Время до следующего манёвра пересчитывается при изменении скорости | ✅ |
| AC-07 | Маршрут строится с учётом велодорожек и лестниц (объезд) | ✅

# Тестирование: негативные сценарии

| № | Сценарий | Ожидаемое поведение |
|---|----------|---------------------|
| 1 | GPS-сигнал пропал на 5 секунд | Система говорит: "Ищу сигнал...", продолжает маршрут по последним данным |
| 2 | Пользователь отклонился от маршрута | Голос: "Вы сбились с маршрута, перестраиваю..." |
| 3 | Нет интернета | Голос: "Нет соединения, маршрут сохранён офлайн" |
| 4 | Шум на улице (ветер, машины) | Голос адаптирует громкость (если поддерживается) |
| 5 | Пользователь отключил голос во время движения | Голос замолкает, маршрут продолжает показываться на экране |
| 6 | Батарея < 10% | Голосовое ведение отключается автоматически с предупреждением

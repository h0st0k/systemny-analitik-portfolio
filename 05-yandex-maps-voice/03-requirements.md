# Требования к голосовому ведению в Яндекс.Картах

## Функциональные требования (FR)

| ID | Требование |
|----|------------|
| FR-01 | Система должна поддерживать построение маршрутов для велосипедов и самокатов |
| FR-02 | Маршрут должен автоматически исключать лестницы и строить по велодорожкам |
| FR-03 | Голосовые подсказки должны активироваться при начале движения по маршруту |
| FR-04 | Алиса должна озвучивать: направление поворота, расстояние до манёвра, оставшееся время |
| FR-05 | Подсказки должны работать в фоновом режиме при свёрнутом приложении |
| FR-06 | Система должна пересчитывать время прибытия при изменении скорости |
| FR-07 | Пользователь может отключить голосовое ведение в любой момент |

## Нефункциональные требования (NFR)

| ID | Требование |
|----|------------|
| NFR-01 | Задержка между сменой позиции и голосовой подсказкой не должна превышать 1 секунду |
| NFR-02 | Приложение должно поддерживать 25 млн активных пользователей без деградации |
| NFR-03 | Фоновый режим не должен разряжать батарею более чем на 10% за час |
| NFR-04 | Голосовые подсказки должны быть понятны даже при внешнем шуме (ветер, город) |

# Кейс: Библиотека — Требования

## FR
| ID | Требование |
|----|------------|
| FR-01 | Показывать каталог с поиском по названию/автору |
| FR-02 | Отображать статус книги (в наличии / на руках / забронирована) |
| FR-03 | Бронировать книгу на срок до 3 дней |
| FR-04 | Не более 3 активных броней на читателя |
| FR-05 | Автоснятие брони через 3 дня |
| FR-06 | История бронирований |

| NFR-02 | Поиск ≤ 2 секунд |

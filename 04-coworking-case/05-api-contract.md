# Кейс: Коворкинг — API-контракт

## GET /desks
Получить список мест с фильтрацией по дате и залу

**Параметры:**
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| date | string | да | Дата в формате YYYY-MM-DD |
| hall_id | integer | нет | ID зала для фильтрации |

**Ответ (200 OK):**
```json
{
  "desks": [
    {
      "id": 1,
      "number": "A12",
      "hall_id": 1,
      "status": "free",
      "description": "Розетка, монитор"
    }
  ]
}
```

## GET /bookings
Получить список броней сотрудника

**Параметры:**
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| employee_id | integer | да | ID сотрудника |

**Ответ (200 OK):**
```json
{
  "bookings": [
    {
      "id": 101,
      "desk_id": 1,
      "booking_date": "2026-06-18",
      "status": "active",
      "created_at": "2026-06-18T10:30:00Z"
    }
  ]
}
```

## POST /bookings
Создать бронь

**Тело запроса:**
```json
{
  "employee_id": 1,
  "desk_id": 2,
  "booking_date": "2026-06-18"
}
```

**Успешный ответ (201 Created):**
```json
{
  "booking_id": 101,
  "status": "active",
  "message": "Место забронировано на 2026-06-18"
}
```

**Ошибка: место уже занято (409 Conflict):**
```json
{
  "error": "Desk already booked on this date",
  "message": "Это место уже забронировано на выбранную дату"
}
```

**Ошибка: превышен горизонт бронирования (400 Bad Request):**
```json
{
  "error": "Booking horizon exceeded",
  "message": "Бронирование возможно не более чем на 2 недели вперёд"
}
```

## DELETE /bookings/{id}
Отменить бронь

**Параметры пути:**
| Параметр | Тип | Описание |
|----------|-----|----------|
| id | integer | ID брони |

**Успешный ответ (200 OK):**
```json
{
  "status": "cancelled",
  "message": "Бронь отменена"
}
```

**Ошибка: бронь не найдена (404 Not Found):**
```json
{
  "error": "Booking not found",
  "message": "Бронь с указанным ID не найдена"
}
```

## POST /bookings/{id}/transfer
Перенос брони (изменение даты или места)

**Параметры пути:**
| Параметр | Тип | Описание |
|----------|-----|----------|
| id | integer | ID брони |

**Тело запроса:**
```json
{
  "new_desk_id": 3,
  "new_date": "2026-06-19"
}
```

**Успешный ответ (200 OK):**
```json
{
  "booking_id": 101,
  "old_desk": 2,
  "new_desk": 3,
  "old_date": "2026-06-18",
  "new_date": "2026-06-19",
  "message": "Бронь перенесена"
}
```

**Ошибка: новое место занято (409 Conflict):**
```json
{
  "error": "Desk already booked on new date",
  "message": "Новое место уже забронировано на эту дату"
}
```

## GET /admin/stats
Получить статистику по бронированиям (только для администратора)

**Ответ (200 OK):**
```json
{
  "total_bookings": 120,
  "used": 95,
  "no_show": 6,
  "cancelled": 19,
  "no_show_percent": 5.0
}
```
